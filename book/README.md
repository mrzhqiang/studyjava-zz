# 理论先行，指导实践

### 入门书籍
事实上，入门应当是具有一定的编程经验，转行加入JAVA语言的行列，或已经被启蒙、被蛊惑，想要成为键盘侠。这里提供的是扫描文档，PDF格式，真正想学习JAVA的你，我建议去购买正版书籍，以备随时随地翻阅。
- [Core Java volume](http://pan.baidu.com/s/1c11N1eg) 《java核心技术》密码：ityb
- [Thinking in java](http://pan.baidu.com/s/1dFfbaud) 《java编程思想》密码：tim0

这两本可以都买回去，参照着看，互相补充营养。建议先看核心技术，入门，然后具有一定的敲代码经验，再回头看编程思想。从书名上，你也能发现它们的不同之处，这里将不做任何比较，因为只有你自己才清楚它们到底有什么不同。

### 提升
当你开发到一定程度，苦恼于技术上的瓶颈，你所需要的正是思想上的提升。
- [Effective Java](http://pan.baidu.com/s/1c2tWkda) 《Effective Java》密码：r646

高效地使用Java，会让你的工作变得更有意义，bug更少，开发心情更好。
