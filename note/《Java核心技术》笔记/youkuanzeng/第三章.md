### （八）控制流程（预感这个会用的特别多，目前只初步了解了一丢，这部分视频我会再看一遍）
- Java 使用条件语句和循环结构确定控制流程。

#### 块作用域
- 由一对大括号括起来的若干条简单的Java语句。
- 块决定了变量的作用域。
- 一个块可以嵌套在另一个块中。但不能在嵌套的两个块中声明同名的变量。

#### 条件语句
- if 条件语句格式：if ( condition ) statement
- 条件必须用括号括起来。
- else子句与最邻近的 if 构成一组。

#### 循环
- 当条件为 true 时， while 循环执行一条语句 （ 也可以是一个语句块 ） 。
- 如果开始循环条件的值就为 false , 则 while 循环体一次也不执行。
-  do / while ：先执行语句 （ 通常是一个语句块 )， 再检测循环条件 ； 然后重复语句，再检测循环条件……

#### 确定循环
- for 循环语句是支持迭代的一种通用结构 ， 利用每次迭代之后更新的计数器或类似的变量来控制迭代次数 。
- 1. 对计数器初始化 ； 2. 给出每次新一轮循环执行前要检测的循环条件 ； 3. 指示如何更新计数.
- 上面3个部分对同一个计数器变量进行初始化 、 检测和更新。
- 如果在 for 语句内部定义一个变量 ， 这个变量就不能在循环体之外使用。
- for 循环语句是 while 循环的一种简化形式。

####  多重选择 ： switch 语句
- switch 语句将从与选项值相匹配的 case 标签处开始执行直到遇到 break 语句， 或者执行到switch 语句的结束处为止。 如果没有相匹配的case 标签 ， 而有 default 子句 ， 就执行这个子句。
- case 标签是：1. 类型为 char 、 byte 、 short 或 int 的常量表达式 ;  2.  枚举常量;  3.   字符串字面量。

####  中断控制流程语句
- goto语句
- 不带标签的 break 语句，可以用于退出循环语句。
- 带标签的 break，用于跳出多重嵌套的循环语句 （**带标签是什么意思呢？**）
- continue语句将控制转移到最内层循环的首部。
