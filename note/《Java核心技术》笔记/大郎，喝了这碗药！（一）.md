# 大郎，喝了这碗药！（一）
转行这种想法，由来已久，在铁路待久了，日复一日，年复一年，光阴虚度，年龄渐长，青葱仿佛往事，24岁，如同老去，种花养狗散步，低头在这屋檐下避了风雨四年，再过几年，我就要在这屋檐下开始猜测你们这些冒雨行进的人去了哪里。有些彷徨，有些惊惧，我知道颓靡腐朽的屋檐迟早倒塌，我不知道雨里的世界会不会有另一番景象。事实上，我虽然厌倦了屋檐下看雨的日子，但是也并不向往传说中“雨后”的彩虹，天知道彩虹后面是不是烈日炎炎。我觉得生来如我这般人，就应该是上帝的视角，看着你们这些卑微的人类，脸上写满怜悯和慈爱。
从来不敢低估接触新事物所要付出的成本，事实上，从过往的经验来看，我还是低估了。你看，重新泡一个妞就要多花不少钱，不管泡妞成功或者失败，投入成本，总会得到教训的，现在讨论投入回报比还没有意义，但是学习抓瓦，对我而言，就像泡妞这事对你而言一样，不管是你泡妞还是妞泡你，无论如何只会是正收益（你可以想象一张嘲讽脸）。当然不管是我学习抓瓦，还是你去泡妞，这都是一个不小的考验。这个考验的关键就是耐性和心态。接着上面说，在这件事情上，对你而言，你要付出的时间和精力成本很高，而且到最后你决定撤资那一刻，很可能收益是负的，这个过程中，你看不到收益率，很多时候会怀疑自己是不是已经裤子都输掉了，怀疑这样的日子什么时候是个头。之后的日子，各种各样的波折肯定不会太少，保持战略上的懈怠战术上的勤奋，是我觉得的最好的安排，你觉得呢？电脑常识的缺乏加上英文上的白痴，开头显得有些许艰难，应该叫琐碎，我感觉经常会卡在一个莫名其妙意想不到的点，进行不下去，这些过往的懈怠，没排掉的雷，只能自己一个一个的踩下去，直至壮烈牺牲，英勇献身，你可以在我的坟前烧一把纸钱，歌功颂德一番，让我得以安眠。学习的过程是充满了“乐趣”的过程，嗯，你认同的对吧？如何让学习变得更有趣，就像是如何让妞变的更主动直至欲罢不能，我想这是作为男人获得满足感和成功的关键，也是需要策划和思考的事情，虽然思考和策划无疑是让你懊恼的。不过，为了你的下半身，哦，下半生，这是不得不做好的事情。当然，无论如何，我都必须给自己做一份学会抓瓦就能让命运女神俯身在我牛仔裤下的毒鸡汤，以保持坚硬和挺拔。当然，如今透过你，看你给我做的学习规划，我感觉我就要看到码农界女神曼妙的姿态了，目前来看，按照你的步骤来，是木有什么问题的。此处应有诗曰：唯有勤恳学抓瓦，不负韶华不负卿。
最后基于这几天的学习情况，友情提示，泡妞有风险，结婚需谨慎。哦，错了，应该是，学习有难度，前路需耐心。最后再对把学习思考总结命名为《大郎，喝了这碗药吧！》予以说明，好吧，那就是没什么好说明的。毕竟良好高效的沟通是减少麻烦和分歧的关键。比如，昨天学的东西，今天忘得干干净净，就需要良好的沟通，好吧，我他吗的是真的忘了。最后，祝好，愿身体健康，平安性福。
