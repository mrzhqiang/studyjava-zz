# 为什么选java？
java是目前最流行的程序开发语言，主要用于后台开发、移动开发、微型设备开发，以及一些桌面应用程序开发。

## 著名应用
- [IntelliJ IDEA](https://www.jetbrains.com/idea/)——最为著名的IDE
- [Eclipse](https://www.eclipse.org/home/index.php)——使用最广泛的IDE
- [Orace DB + JDK](https://www.oracle.com/cn/downloads/index.html)——最强势的组合
- Android大部分是用java开发
- 目前最流行的web端也是基于java语言

## 总结
- 最流行的开发语言，开源库很多，开发时能节省很多精力
- 应用范围广泛，就业形势良好
- 面向对象，适合上手，开发项目舒服
- 比较接近自然语言，阅读体验很棒
- 拥有大量开发人员，能够协同工作
