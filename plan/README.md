# 备课
没有备课的话，讲解课程就显得随意，对于学习者来说，就是一场灾难。备课代表侧重点，代表该讲什么，不该讲什么，重点讲什么，什么需要一笔带过。良好的备课应该控制好时间，在合适的时间里，讲出最恰当的内容，不多不少，深入浅出，言简意赅。
